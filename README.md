##### `git clone https://github.com/LucasViana1/banco-pessoas.git`

Instale as dependÃªncias utilizando [yarn](https://classic.yarnpkg.com/en/docs/install/#windows-stable) ou [npm](https://www.npmjs.com/get-npm):

Execute o projeto em desenvolvimento:

##### `npm run start`

##### `npm run backend`
